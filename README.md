# sistema-tickets

## Pasar esta versión al branch principal (`main`)

1. Abrí el Pull Request de este branch en GitHub.
2. Hacé clic en **Merge pull request** para integrar los cambios en `main`.
3. Verificá que el commit aparezca en la rama `main`.

## Traer `main` a tu localhost y probar

Si ya tenés el repo clonado:

```bash
git checkout main
git pull origin main
```

Si todavía no lo tenés clonado:

```bash
git clone https://github.com/metorres-ulp/sistema-tickets.git
cd sistema-tickets
git checkout main
```

Con eso ya vas a tener en local la versión más reciente de `main`.
